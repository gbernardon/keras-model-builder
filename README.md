# Keras Model Builder
Image classification model builder using Keras and TensorFlow.

### Required Modules:
- Keras 2.4.3
- TensorFlow 2.5.0
- Numpy 1.19.2
- Six 1.15.0
- Pillow 8.2.0
- Matplotlib 3.4.2
- os
